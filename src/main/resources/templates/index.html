• - /api/auth:

    GET: /user -> Tokenı sorgular, kullanıcı adını dönderiyor.

    POST: /login body:authrequest-> token dönderiyor.

    POST: /register body:user-> token dönderiyor.

  • - /api/book_user:

    GET: /{id} -> user dönderiyor.

    POST: body:user

    PATCH: /{id} body:transferUser

    DELETE: /{id}

  • - /api/search:

    GET: /name/{name} -> Ismi içeren kullanıcıları dönderiyor.

    GET: /department/{department} -> Departmanı içeren kullanıcıyı dönderiyor.

  • - /api/timetable:

    GET: /{userId} -> userId ye bağlı olan timetable yapılarını dönderiyor.

    POST: /{userId} body:timetable

    PATCH: /{tableId} body:timetable

    DELETE: /{userId}

  1. authrequest = {username, password}
  2. user = {username, password, department, position, phoneNumber:integer, email, officeNo, timetableListesi}
  3. transferUser = {username, department, position, phoneNumber:integer, email, officeNo}
  4. timetable = {dayOfWeek:integer, hour, className}
